# University Course Management System

A database-driven system designed to manage university courses, students, and enrollments efficiently.  
This project demonstrates the implementation of core **Database Management System (DBMS)** concepts such as data storage, retrieval, and management.

## Features

- Add and manage student records
- Add and manage course information
- Enroll students in courses
- View enrolled courses for each student
- Search and retrieve student/course details
- Update and delete records
